# Defense Proposal — 세부 연구 계획 스킬

방산 연구 제안서의 **세부 연구 계획** 섹션(6개 항목)을 작성한다.
이 섹션은 예산·자원·시험·안전·성과물에 대한 실행 가능성과 구체성을 보여준다.

## 섹션 구성

| 항 | 제목 | 핵심 내용 |
|---|---|---|
| 1 | 과제 소요예산 | 비목별 예산 상세 내역 |
| 2 | 세부과제별 소요예산 총괄 | 세부과제별 예산 배분 |
| 3 | 소요자원 및 활용 계획 | 인력, 장비, 시설 계획 |
| 4 | 성능시험 계획 | 시험 항목, 방법, 기준 |
| 5 | 안전관리 계획 | 위험 요소 및 대응 방안 |
| 6 | 연구개발 성과물 | 납품 목록 및 특허·논문 계획 |

---

## Step 1 — 기본 정보 확인

- 총 예산 및 연차별 배분
- 참여 인력 구성 (연구책임자, 연구원 수, 외부 인력)
- 주요 장비·시설 보유 여부 및 구매 계획
- 예상 시험 환경 (실내/야외/모의/실환경)
- 방산 특수 안전 항목 (폭발물, 고압, 전자파 등)

---

## Step 2 — 항목별 콘텐츠 작성

### 2-1. 과제 소요예산 (1항)

**비목별 예산 내역 표 (연차별):**

|---|---|---|---|---|---|
| **인건비** | | | | | |
| 연구책임자 | 월 X백만원 × 12개월 × Y% | | | | |
| 연구원 | 인원수 × 단가 × 투입률 | | | | |
| **직접비** | | | | | |
| 재료비 | | | | | |
| 전산처리비 | | | | | |
| 시험분석비 | | | | | |
| 외주용역비 | | | | | |
| 특허출원비 | | | | | |
| **간접비** | 직접비 × 요율 | | | | |

**산출 근거 명시 원칙:**
- 모든 비목은 단가 × 수량 × 투입율 형식으로 근거 작성
- "일식" 표현 최소화 — 구체적 수량 명시

### 2-3. 소요자원 및 활용 계획 (3항)

**인력 계획:**
| 구분 | 성명/직급 | 전문 분야 | 연차별 투입률 | 담당 역할 |
|---|---|---|---|---|
| 연구책임자 | | | 30% / 30% / 30% | |
| 연구원 1 | | | 50% / 50% / 50% | |

**장비/시설 계획:**
| 장비명 | 보유/구매 | 용도 | 활용 시기 | 단가(백만원) |
|---|---|---|---|---|
| [장비명] | 구매 | [용도] | 1차년도 | X |

**외부 자원 활용:**

**슬라이드**: 인력 투입 계획 표 (1장) + 장비/시설 계획 표 (1장)

### 2-4. 성능시험 계획 (4항)

**시험 체계 계층:**
```
시스템 시험 (최종)
  ├── 서브시스템 통합 시험
  │   ├── 구성품 단위 시험
  │   └── 인터페이스 시험
  └── 환경 시험 (기계적, 전기적, 기후 등)
```

**시험 항목별 계획표:**
| 시험 항목 | 시험 방법 | 합격 기준 | 시험 시기 | 시험 장소 |
|---|---|---|---|---|
| [성능 항목 1] | [방법] | ≥ X [단위] | 2차년 Q3 | 자체 시험장 |

**단계별 시험 절차:**
1. 컴퓨터 시뮬레이션 검증
2. 랩 레벨 구성품 시험
3. 시제품 통합 시험
4. 실환경 현장 시험

**defense-diagram 스킬**로 시험 절차 플로우차트(F) 생성.

**슬라이드**: 시험 체계도 (1장) + 시험 항목표 (1–2장) + 시험 절차 (1장)

### 2-5. 안전관리 계획 (5항)

**위험 요소 식별:**
방산 연구의 특수 위험 요소를 점검:
- 화약류/폭발물 취급 여부
- 고전압/고전력 장비
- 전자파(RF) 방사
- 레이저 등 광학 위험
- 유해 화학물질
- 고압 가스

**위험도 매트릭스:**
| 위험 요소 | 발생 가능성 | 영향도 | 위험 등급 | 대응 방안 |
|---|---|---|---|---|
| [위험 요소] | 高/中/低 | 高/中/低 | 高/中/低 | [방안] |

**안전 관리 체계:**
- 안전관리 담당자 및 비상 연락 체계
- 안전 교육 계획 (주기, 내용)
- 비상 대응 절차

**관련 법규 및 기준:**
- 적용 법규 명시 (산안법, 방사선, 화학물질 등)

**슬라이드**: 위험도 매트릭스 (1장) + 안전 관리 체계 (1장)

## Step 3 — PPTX 슬라이드 생성

**슬라이드 순서:**
```
[섹션 표지] "세부 연구 계획"
  → 1. 과제 소요예산 (2–3장: 비목별 내역)
  → 2. 세부과제별 소요예산 총괄 (1–2장: 표 + 차트)
  → 3. 소요자원 및 활용 계획 (2장: 인력 + 장비)
  → 4. 성능시험 계획 (3장: 체계도 + 항목표 + 절차)
  → 5. 안전관리 계획 (2장)
  → 6. 연구개발 성과물 (2장)
```

## 작성 품질 기준

- 예산 합계: 반드시 수식으로 검산 (단순 합산 오류 방지)
- 인력 투입률: 합계가 100%를 초과하면 안 됨 (과부하 점검)
- 시험 합격 기준: 과제 개요의 성능 목표치와 정확히 일치
- 안전 계획: 방산 특수 위험 요소 빠짐없이 점검
- 성과물: 납품 목록은 계약서 수준의 구체성
