## I/O Type Compatibility Table

| Source Type | Target Type | Compatible | Conversion |
|---|---|:---:|---|
| `np.ndarray(float32)` | `np.ndarray(float32)` | ✅ | — |
| `np.ndarray(float64)` | `np.ndarray(float32)` | ⚠️ | `.astype(np.float32)` |
| `np.ndarray` | `torch.Tensor` | ❌ | `torch.from_numpy(x)` |
| `torch.Tensor` | `np.ndarray` | ❌ | `x.detach().cpu().numpy()` |
| `List[float]` | `np.ndarray` | ⚠️ | `np.array(x, dtype=np.float32)` |

## Algorithm Summary → Code Pattern Mapping

| Algorithm Summary Keyword | Code Pattern |
|---|---|

| Convolution | `scipy.signal.convolve`, `torch.nn.functional.conv1d` |
| Normalization (L2) | `x / (np.linalg.norm(x) + eps)` |
| Matrix multiplication | `x @ W`, `np.einsum` |
| Moving average / smoothing | `np.convolve(x, np.ones(k)/k, mode='same')` |
| Thresholding / clipping | `np.clip(x, lo, hi)` |
| Interpolation | `scipy.interpolate.interp1d` |
| Sort / rank | `np.argsort`, `np.argpartition` |
