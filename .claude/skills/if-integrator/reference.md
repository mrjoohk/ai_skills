# Reference: IF Integrator

## Project Layout Convention

```
src/
  uf/
    <uf_module>.py          ← Unit function implementations (input to if-integrator)
  if/
    <if_module>.py          ← IF integration modules (output of if-integrator)
tests/
  unit/
    test_<uf_module>.py     ← UF-level tests (from uf-implementor)
  integration/
    test_<if_module>.py     ← IF-level integration tests (output of if-integrator)
reports/
  impl/
    if_integration_report_<timestamp>.md
```

## IF Integration Module Template

```python
#!/usr/bin/env python3
"""
Module: <if_module_name>
Integration Function: IF-##  <IF Goal>
Constituent UFs: UF-01, UF-02, UF-03
REQ Links: REQ-01, REQ-04

Generated by: if-integrator
Date: <date>
Source: if_list.md, if_decomposition.md
"""

from __future__ import annotations
import numpy as np
from src.uf.<uf_module> import uf_01_<name>, uf_02_<name>, uf_03_<name>


class IntegrationError(RuntimeError):
    """Raised when a UF postcondition is not satisfied during IF execution."""
    pass


def if_<id>_<snake_name>(
    <if_input>: np.ndarray,   # shape: (<dims>), unit: <unit>
    **kwargs,
) -> np.ndarray:              # shape: (<dims>), unit: <unit>
    """<IF Goal from if_list.md>

    Integration sequence: UF-01 → UF-02 → UF-03
    REQ links: REQ-01, REQ-04

    Args:
        <if_input>: <description>. Shape: (<dims>,). Unit: <unit>.

    Returns:
        <if_output>: <description>. Shape: (<dims>,). Unit: <unit>.

    Raises:
        IntegrationError: if any UF postcondition fails.
        NotImplementedError: if a required UF is STUB or BLOCKED.
    """
    # --- UF-01: <name> ---
    stage_1 = uf_01_<name>(<if_input>)

    # --- UF-02: <name> (depends on UF-01 output) ---
    stage_2 = uf_02_<name>(stage_1)

    # --- UF-03: <name> (depends on UF-02 output) ---
    result = uf_03_<name>(stage_2)

    # --- Postcondition check (IF acceptance criteria) ---
    _check_if_postcondition(result, threshold=<threshold>)

    return result


def _check_if_postcondition(result: np.ndarray, threshold: float) -> None:
    """Verify IF-level acceptance criterion. Raise IntegrationError on failure."""
    metric = _compute_metric(result)
    if metric < threshold:
        raise IntegrationError(
            f"IF postcondition failed: metric={metric:.4f} < threshold={threshold}"
        )
```

## UF Call Orchestration Patterns

### Sequential (linear pipeline)
```python
x1 = uf_01(input)
x2 = uf_02(x1)
x3 = uf_03(x2)
return x3
```

### Parallel (independent branches, merge)
```python
import concurrent.futures

with concurrent.futures.ThreadPoolExecutor() as executor:
    f_a = executor.submit(uf_02a, x1)
    f_b = executor.submit(uf_02b, x1)
x2a, x2b = f_a.result(), f_b.result()
return uf_03_merge(x2a, x2b)
```

### Conditional branch
```python
x1 = uf_01(input)
if condition(x1):
    return uf_02a(x1)   # fast path
else:
    return uf_02b(x1)   # fallback path
```

### Iterative / feedback loop
```python
state = init_state(input)
for step in range(max_steps):
    state = uf_02_update(state)
    if convergence_check(state):
        break
return uf_03_finalize(state)
```

## IF Integration Status Codes

| Code | Meaning |
|---|---|
| `COMPLETE` | All UFs integrated, all IF acceptance criteria passed |
| `PARTIAL` | Integration module generated; some tests fail or UFs are STUB |
| `INTERFACE_ERROR` | UF signature mismatch with IF I/O contract; blocked |
| `BLOCKED` | Required UF is STUB/BLOCKED; IF raises NotImplementedError at runtime |

## REQ ↔ IF ↔ UF Traceability Matrix Format

```markdown
| REQ-ID | Criterion | IF | UF Chain | Status |
|---|---|---|---|:---:|
| REQ-01 | Accuracy ≥ 95% | IF-01 | UF-01→UF-02→UF-03 | ✅ PASS |
| REQ-02 | Latency ≤ 50ms | IF-01 | UF-01→UF-02→UF-03 | ❌ FAIL |
| REQ-03 | Memory ≤ 1 GB  | IF-02 | UF-04→UF-05        | ✅ PASS |
```
