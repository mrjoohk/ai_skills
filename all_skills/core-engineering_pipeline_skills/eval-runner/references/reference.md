```python

    accuracy_score, f1_score, roc_auc_score,
    precision_recall_curve, matthews_corrcoef
```

## Statistical Significance Reporting (if needed)

```python
# Bootstrap confidence interval
import numpy as np

def bootstrap_ci(scores, n_bootstrap=1000, ci=0.95):
    bootstraps = [
        np.mean(np.random.choice(scores, size=len(scores), replace=True))
        for _ in range(n_bootstrap)
    ]
    lo = np.percentile(bootstraps, (1 - ci) / 2 * 100)
    hi = np.percentile(bootstraps, (1 + ci) / 2 * 100)
    return np.mean(scores), lo, hi

# Result notation: 12.8 ± 0.3 dB (95% CI: [12.5, 13.1])
```
